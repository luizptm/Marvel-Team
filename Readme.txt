1.	Conectar com a API da MARVEL. https://developer.marvel.com/
2.	Trazer 2 listas aleatórias com 10 personagens em cada lado da tela.
2.1.	Não trazer personagens com descrição vazia ou sem nome 
2.2.	Exibir a imagem deles em uma tela com seus nomes.
3.	Permitir que o usuário monte um time com personagens de cada lista clicando em seus retratos ou nomes, com até 3 personagem em cada time.
4.	O nível de força de cada personagem será a quantidade de histórias (quadrinhos) em que o personagem participou/apareceu.
5.	Comparar o nível de poder de combate das duas equipes, onde a equipe com maior poder de combate vence.
6.	Salvar os resultados em um banco de dados (Servidor ou LocalStorage) o Time vitorioso, integrantes de cada time, e potencial de cada personagem o time.
7.	Ao fim os vencedores devem ser exibidos e abaixo um resumo da história de cada um e suas imagens.

Regras:
# Pode ser utilizado ASP.NET Webform; Asp.Net MVC; WCF ou Web Api;
# Permitido utilizar qualquer biblioteca de terceiro ou própria, inclusive para acessar a API Marvel;

# Utilizar IDE Visual Studio 2012 ou mais atual;
